## LaTeX 문법 사용법 (MathJax/KaTeX 기준)

LaTeX 문법은 크게 **인라인(Inline)** 방식과 **블록(Block)** 방식으로 나뉩니다.

* **인라인 방식**: 문장 중간에 수식을 넣을 때 사용합니다.
    * 기본 문법: `$수식$` 또는 `\\(수식\\)`
    * 예시: `이차 방정식 $ax^2 + bx + c = 0$의 해는 ...`

* **블록 방식**: 수식을 별도의 줄에 중앙 정렬하여 강조할 때 사용합니다.
    * 기본 문법: `$$수식$$` 또는 `\\[수식\\]`
    * 예시: `피타고라스의 정리는 다음과 같습니다. $$a^2 + b^2 = c^2$$`

## 자주 사용되는 LaTeX 기호 및 문법 정리

| 기능             | 문법                                  | 예시                                             |
| :------------- | :---------------------------------- | :--------------------------------------------- |
| **분수**         | `\frac{분자}{분모}`                     | `\frac{1}{2}`                                  |
| **제곱**         | `a^b`                               | `x^2`                                          |
| **아랫첨자**       | `a_b`                               | `x_i`                                          |
| **제곱/아랫첨자 혼합** | `a^b_c`                             | `x_i^2`                                        |
| **제곱근**        | `\sqrt{수}`                          | `\sqrt{x^2+y^2}`                               |
| **합계**         | `\sum_{시작}^{끝}`                     | `\sum_{i=1}^{n}`                               |
| **적분**         | `\int_{시작}^{끝}`                     | `\int_{0}^{1} x^2 dx`                          |
| **무한대**        | `\infty`                            | `\infty`                                       |
| **미분 기호**      | `\partial`                          | `\frac{\partial f}{\partial x}`                |
| **벡터 기호**      | `\vec{변수}`                          | `\vec{v}`                                      |
| **그리스 문자**     | `\alpha`, `\beta`, `\gamma` 등       | `\alpha`, `\beta`, `\pi`                       |
| **함수**         | `\sin`, `\cos`, `\log` 등            | `\sin(\theta)`                                 |
| **점**          | `\dots` (중간), `\ldots` (아래)         | `1, 2, \dots, n`                               |
| **화살표**        | `\to`, `\implies` 등                 | `A \to B`, `A \implies B`                      |
| **행렬**         | `\begin{pmatrix} ... \end{pmatrix}` | `\begin{pmatrix} 1 & 2 \\ 3 & 4 \end{pmatrix}` |
| **띄어쓰기**       | `\,`, `\;`, `\quad` 등               | `x \, y`                                       |

## 명제 관련

| 기호  | 설명          | LaTeX 명령어         |
| --- | ----------- | ----------------- |
| ∈   | 원소          | `\in`             |
| ∉   | 원소가 아님      | `\notin`          |
| ⇒   | 함의          | `\Rightarrow`     |
| ⇔   | 동치          | `\Leftrightarrow` |
| ↔   | 동치(양방향 화살표) | `\leftrightarrow` |
| ⊢   | 증명 가능       | `\vdash`          |
| ⊨   | 논리적 귀결      | `\models`         |
| ∀   | 모든          | `\forall`         |
| ∃   | 존재          | `\exists`         |
| ⊆   | 부분집합        | `\subseteq`       |
| ⊂   | 진부분집합       | `\subset`         |
| ∪   | 합집합         | `\cup`            |
| ∩   | 교집합         | `\cap`            |
| ∧   | 논리곱         | `\wedge`          |


## 주의사항

* **환경 설정 확인**: 사용 중인 마크다운 렌더러가 LaTeX 문법을 지원하는지 확인해야 합니다.
* **이스케이프(Escape)**: `$` 기호를 문자로 사용하고 싶다면 `\$`와 같이 앞에 `\`를 붙여야 합니다.
* **중괄호 `{}` 사용**: 첨자나 분수 등 여러 글자에 영향을 미치게 할 때는 중괄호로 범위를 지정해야 합니다. 예를 들어, `x^10`을 표현하려면 `x^{10}`으로 작성해야 합니다.
