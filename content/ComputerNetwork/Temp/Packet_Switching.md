# Computer Network - Packet Switching


## 패킷 스위칭

긴 메시지를 **"패킷"** 이라는 데이터 덩어리로 나누어 전달한다.

**Circuit Switching (회선 교환)** 에서 통신 세션이 유지되는 동안 경로상의 자원 (버퍼, 링크 전송률 등) 이 해당 연결만을 위해 할당되는 것과는 다르게

**Packet Switching** 에서는 자원을 미리 예약해놓지 않고 필요할 때만 자원을 사용한다. (**On-demand**)
